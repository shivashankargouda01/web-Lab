HTML Basics
HTML (HyperText Markup Language) is the standard markup language for creating web pages. It provides the structure of a webpage, defining the content and its organization.

Key concepts include:
  • Tags: Elements are defined by tags, such as <p> for paragraphs and <h1> for headings.
  • Attributes: Tags can have attributes that provide additional information, like <img src="image.jpg" alt="description">.
  • Document Structure: An HTML document typically includes <html>, <head>, and <body> sections.

Understanding the basic structure is essential for building web pages.
HTML Forms
HTML forms are used to collect user input. They consist of various input elements like text fields, checkboxes, and buttons.

Common form elements:
  • <input>: Used for various input types, like text, password, and radio buttons.
  • <textarea>: Used for multi-line text input.
  • <select>: Used for dropdown lists.
  • <button>: Used to submit or perform actions.
HTML Tables
HTML tables are used to display data in a structured format. They consist of rows and columns.

Key table elements:
  • <table>: Defines the table.
  • <tr>: Defines a table row.
  • <th>: Defines a table header cell.
  • <td>: Defines a table data cell.
HTML Media
HTML allows embedding media content like images, videos, and audio into web pages.

Common media elements:
  • <img>: Used to embed images.
  • <video>: Used to embed videos.
  • <audio>: Used to embed audio.
  • <iframe>: Used to embed other web pages or media.
